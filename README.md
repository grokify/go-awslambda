## Overview

`awslambda` is helper package for AWS Lambda on Go.

See the docs on [pkg.go.dev](https://pkg.go.dev/github.com/grokify/go-awslambda).

## Installation

```bash
$ go get github.com/grokify/go-awslambda/...
```
